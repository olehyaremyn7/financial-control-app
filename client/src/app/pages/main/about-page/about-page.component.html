Aligator Money

This web application is designed to control the finances of users. The functionality of the application is that the user creates accounts for the desired purpose, specifying the balance of each of them, and then enters the transactions into the appropriate account. Transactions can reflect a user's expenses and income, and their impact on the overall account balance, showing the amount of expenses and income relative to the current balance.

This web application is a demonstration of the use of modern web technologies to develop powerful one-page web applications based on the JavaScript programming language. Technology stack: Angular, NodeJS, ExpressJS, MongoDB.

The idea of this web application was borrowed from popular applications for cost control. The essence of this web application is not commercial, but for the purpose of the practice of building complex web applications.
