Aligator Money

This web application is designed to control the finances of users. The functionality of the application is that the
user creates accounts for the desired purpose, specifying the balance of each of them, and then enters the transactions
into the appropriate account. Transactions can reflect a user's expenses and income, and their impact on the overall
account balance, showing the amount of expenses and income relative to the current balance.

To start using the web application, you need to register and then log in. After that, you can read more about this
project on the About page.
